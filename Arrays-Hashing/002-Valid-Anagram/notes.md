# 📝 Notes

## 🏷️ Pattern

Frequency Counting

## 🚀 Optimized Solution

Use a HashMap to count character occurrences.

## 🐌 Brute Force

Sort both strings and compare.

Time Complexity: O(n log n)

## ⚠️ Common Mistakes

- Forgetting to compare lengths first.
- Ignoring character frequencies.
- Not handling repeated characters.

## 🔄 Similar Problems

- Group Anagrams
- Find All Anagrams in a String
- Ransom Note

## 🎯 Pattern Recognition

Whenever you see:

- Character counts
- Frequency comparisons
- Rearranging letters

Think HashMap or frequency array.
