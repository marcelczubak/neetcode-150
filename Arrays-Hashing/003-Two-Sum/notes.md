# 📝 Notes

## 🏷️ Pattern

Complement Lookup

## 🚀 Optimized Solution

Store numbers and indices in a HashMap.

Lookup complement before inserting current value.

## 🐌 Brute Force

Check every pair.

Time Complexity: O(n²)

## ⚠️ Common Mistakes

- Returning values instead of indices.
- Adding current number before checking complement.
- Forgetting duplicates can exist.

## 🎯 Pattern Recognition

If a problem asks:

- Find two values
- Find a target sum
- Find complements

Think HashMap.
